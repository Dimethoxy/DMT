\doxysubsection{Detailed Description}
Represents a custom arc-\/shaped button component. 

The \mbox{\hyperlink{classdmt_1_1_arc_button_component}{Arc\+Button\+Component}} class is a custom component that displays an arc-\/shaped button. It inherits from the juce\+::\+Component class and implements the juce\+::\+Mouse\+Listener interface. The button can be left-\/sided or right-\/sided based on the constructor parameter.
