Detailed Description

Represents a custom arc-shaped button component.

The ArcButtonComponent class is a custom component that displays an arc-shaped button. It inherits from the juce::Component class and implements the juce::MouseListener interface. The button can be left-sided or right-sided based on the constructor parameter.
